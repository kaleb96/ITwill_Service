서버(Server) : 사용자로부터 정보를 요청받아 제공하는 측
클라이언트(Client) : 서버에게 정보를 요청하는(제공받는) 측

Client-Side Language : HTML, CSS, Javascript
=> 클라이언트측에서 프로그램을 번역 및 실행하는 언어

Server-Side Language : JSP, ASP, PHP
=> 서버측에서 프로그램을 번역 및 실행 후 결과를 클라이언트에게 전송
