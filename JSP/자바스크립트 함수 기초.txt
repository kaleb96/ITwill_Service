함수(function)
- 실행할 코드의 묶음
- 자바스크립트에서 미리 작성하여 제공하는 함수를 내장함수라고 함

< 함수 사용 기본 문법 >
1. 함수를 호출만 하고 아무런 데이터도 전달하지 않는 경우
함수명();

2. 함수 호출할 때 해당 함수에서 사용할 데이터를 전달하는 경우
함수명(데이터);
=> 전달할 데이터는 단일 또는 복수개의 데이터 전달이 가능하며
   복수개의 데이터일 경우 콤마(,)로 구분함
ex) alert("함수호출"); 
    => alert() 함수에 "함수호출" 문자열 전달하고
       전달받은 문자열을 화면에 출력하는 역할 수행
