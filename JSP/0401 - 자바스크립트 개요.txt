Javascript(자바스크립트)
- 생성된 웹페이지에 사용자가 액션을 취했을 때
  해당 액션에 대한 반응을 수행할 수 있도록 해주는 언어
  (= 웹페이지에 생동감을 불어넣을 수 있도록 해주는 프로그래밍 언어)
  ex) 회원가입 버튼 클릭 시 버튼 클릭에 대한 동작
      패스워드 입력 시 입력된 패스워드가 규칙에 맞는지 체크
- 자바스크립트로 작성된 프로그램을 스크립트라고 하며
  자바스크립트를 스크립트 언어라고 한다.
  => 스크립트 언어는 별도의 번역(컴파일)없이 일반 문자 형태로 작성하고 바로 실행이 가능함
     따라서, 자바스크립트는 일반 텍스트 문서로 작성하고, 웹브라우저에서 즉시 실행됨
- ECMAScript 언어로도 불림(ex. ES6 등)
- 자바와 아무런 관계가 없는 언어이지만, 자바와 비슷한 문법을 많이 사용함
- HTML 문서 수정, 마우스나 키보드 등의 동작(이벤트) 반응 처리 등의 작업을 수행 가능
- 대부분의 웹브라우저에서 지원
- 객체 기반 언어이며 무료로 공개되어 있고, 라이브러리가 다양
- 자바스크립트에서 제공되는 내장 객체들과 다양한 함수들을 활용하여
  웹페이지에 다양한 부가적인 작업 수행 가능
  ex) document 객체, alert() 함수 등


자바스크립트를 사용하는 이유
- 차후 JQuery(AJAX 포함) 등을 활용하는데 필요
- 클라이언트(웹브라우저)에서 실행되므로 서버에 부하가 적게 걸림
- 정적인 웹사이트 -> 동적인 웹사이트로 변환 가능


자바스크립트 기본 사용법
- HTML 문서 내에서 <script></script> 태그를 사용하여 자바스크립트 영역 설정
