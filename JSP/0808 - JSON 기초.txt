JSON(JavaScript Object Notation)
- 다양한 플랫폼에서 대량의 데이터를 주고 받기 위해 사용 가능한 데이터 교환 포맷
- 원래, 자바스크립트 내에서 데이터를 교환하기 위한 포맷으로 개발되었으나
  현재는 플랫폼 관계없이 다양한 언어에서 사용
- JSON 은 별도의 파싱이나 직렬화 없이도 자바스크립트에서 사용 가능하며
  객체, 리터럴, 배열 등의 다양한 데이터를 표현할 수 있는 텍스트 기반 포맷
  (문자열, 숫자, boolean, Null, 객체, 배열)
- 네트워크를 통해 대량의 데이터를 전송하는데 용이함
- JSON 에서 객체는 중괄호{} 를 사용하여 표기하며, 중괄호 내에 "속성명":값" 형태로 표기
  배열은 대괄호 표기하며, 대괄호 내에 값을 직접 지정하거나 객체를 추가할 수 있음

< JSON 데이터 작성 기본 문법 >
기본적으로 데이터는 "속성명":값 의 형태로 나타내며
객체를 의미하는 중괄호{} 내에 데이터를 작성함
ex)
{
	"속성명1":데이터1,
	... 생략 ...
	"속성명n":데이터n
}


< 복수개의 JSON 객체를 저장하는 배열 사용 기본 문법 >
기본적으로 배열을 의미하는 대괄호[] 를 표시한 후 대괄호 내에
객체를 의미하는 복수개의 중괄호{}를 콤마(,)를 사용하여 구분 후 나열
ex)
[
	{"객체1속성명1":객체1데이터1, ..., "객체1의속성명n":객체1의데이터n},
	... 생략 ...
	{"객체x속성명1":객체x데이터1, ..., "객체x의속성명n":객체x의데이터n},
]
