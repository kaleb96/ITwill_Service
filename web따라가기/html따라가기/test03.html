목록 만들기

목록의 종류

  • 순서 없는 목록 - <ul>
  • 순서 있는 목록 - <ol>
  • 정의 목록 - <dl>

[ ♣ 특수문자 태그들 ] - &로 시작하고 ;로 끝남

  • 결과 = 태그(명령)
  • < = &lt;
  • > = &gt;
  •   = &nbsp; (문자 하나의 공백-문자 높이를 가짐)
  • & = &amp;

(ex) <b>
<!-- 주석태그닷 -->

  • © = &copy;
  • ™ = &trade;
  • ® = &reg;
