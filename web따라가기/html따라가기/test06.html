링크 걸린 텍스트의 특징

  • 색상 변화
    1. blue - 초기 링크. (이동한 적이 없는 주소)
    2. red - 활성화 링크. (마우스커서로 누르는 동안)
    3. violet - 방문한 링크. (방문한 적이 있는 주소)
  • 밑줄
  • 커서가 손모양으로 바뀜

링크를 여는 방식 지정 - target="_self/_blank/..."

  1. _self - 기본값. 현재 위치에 링크가 열림
  2. _blank - 새 탭에 링크가 열림
  3. _parent - 상위창에 링크가 열림
  4. _top - (프레임구조에서) 나뉨없이 전체에 하나로 링크가 열림
